# Terros Connect integration template

Start here when building a Connect integration

This repository contains a prebuilt development environment to get you building an integration as quickly as possible

All the following comes preconfigured:

- Terros SDK preinstalled
- Preset integration stub functions
- GitHub Actions CI pipeline
- Automated release deployment to Terros (coming soon)

## Getting Started

1. Duplicate this repository template by clicking "Use this template"
2. Clone your new repository to your computer
3. Run `pnpm install` to install the required dependencies
4. Make changes to `src/index.ts` to get started
